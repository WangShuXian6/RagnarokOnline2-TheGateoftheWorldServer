this method is used for adding a personal quest to a npc, analog to AddQuest

 

public void AddPersonalQuest(uint qid, uint clv, uint previous, uint script)

 

qid: QuestID

clv: CLv of player required in order to get this quest

previous: Previous quest reqiured for this quest

script: chatting script for this personal quest

 

 

Remark: the script parameter in this method is very important, you must use the right number, or you'll not able to get the quest, it's predefined in client!
